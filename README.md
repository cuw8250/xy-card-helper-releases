# XY-填卡助手(等待发布)

一款集成多个邮箱系统的浏览器扩展，支持 Cursor、Augment 平台的自动填入虚拟卡及地址信息。

## ✨ 主要功能

- **🔐 多邮箱系统集成** - 支持多个主流邮箱服务
- **📧 自定义域名邮箱** - 支持配置自定义域名邮箱服务
- **💳 自动填卡** - 一键自动填写虚拟卡、地址信息到 Cursor、Augment 绑卡页
- **🚀 自动注册** - 自动完成账号注册流程
- **⚡ 一键操作** - 简洁的界面，便捷的操作体验

## 📥 下载安装

### 方式 1：拖拽安装（推荐）

1. 访问 [Releases](https://github.com/cuw8250/xy-card-helper-releases/releases) 页面
2. 下载最新版本的 ZIP 文件（**无需解压**）
3. 打开 Chrome 浏览器，访问 `chrome://extensions/`
4. 开启"开发者模式"
5. 将 ZIP 文件直接拖拽到浏览器窗口中

### 方式 2：解压安装

1. 访问 [Releases](https://github.com/cuw8250/xy-card-helper-releases/releases) 页面
2. 下载最新版本的 ZIP 文件并解压
3. 打开 Chrome 浏览器，访问 `chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的文件夹
